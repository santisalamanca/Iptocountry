How to use Iptocountry:
1) Copy files to your hosting/website
2) Create folder "countries"
3) Call/Run/Execute ipblockgenerator.php
4) Verify creation of files within folder countries
5) Test iptocountry.php?ip=xxx.xxx.xxx.xxx    (Replace xxx.xxx.xxx.xxx with a real IP)

Advantage 
- Execution is very fast
- You don't need a DataBase
- You can add security layers
- You don't depend on external servers
- No limitations in queries per minute/hour/day/week
- Minimum Data Transfer
